# Codex Notes

This folder centralizes Codex-generated project support artifacts so the repo root stays readable.

## Layout

- `handoff/`
  - current handoff / stopping-point document
- `logs/`
  - timestamped running review log
- `memory/`
  - branch memory / readiness notes
- `notes/`
  - one-off assessment, audit, and planning notes
- `plans/`
  - longer-form implementation planning documents
