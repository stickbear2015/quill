# AI Writing Skills

A bundled QUILL Quillin that ships four multi-step AI writing skills (`.sqp` files) for the QUILL Skill Library.

## Included skills

### Accessible Rewrite

Analyses selected text for plain-language issues (sentence length, passive voice, jargon, abstract nouns), then produces a targeted rewrite that preserves all factual content. Supports selectable reading levels (Grade 6/8/10).

### Research and Draft

Extracts the central topic from your selection, gathers five supporting facts, then drafts a full paragraph at your chosen tone and length. Good for expanding a topic sentence into a complete paragraph.

### Meeting Notes to Action Items

Reads meeting notes, extracts every action item with owner and deadline, checks for missed items, and produces a clean follow-up summary ready to share. Output goes to clipboard.

### Argument Strengthener

Identifies the logical structure of an argument, finds weaknesses and counterarguments, then produces a strengthened version tailored to your target audience (general public, academic, professional, skeptical expert).

## File format

Each skill is a `.sqp` (Skill Quill Pack) file: a Markdown document with YAML front matter. Open any `.sqp` file in QUILL to read or edit it directly — no tooling required.

## Usage

Skills load automatically when you open the Skill Library (`Tools > AI Assistant > Skill Library`). Select a skill, review its steps and parameters, fill in your inputs, and run.
